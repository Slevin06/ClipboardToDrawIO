ClipboardToDrawIO

クリップボードにコピーしたDraw.io形式のXMLデータからDrawioファイルを簡単に作成できるアプリです。

インストール手順 使用方法 GitHubリポジトリ

概要

ClipboardToDrawIOは、AIアシスタントなどから生成されたDraw.io形式のXMLをクリップボードからDrawioファイルとして保存するmacOS用アプリケーションです。

主な機能

  • クリップボードからDraw.io XMLを検出
  • ワンクリックでDrawioファイルを作成
  • カスタム保存先とファイル名の指定
  • XMLヘッダーの自動修正
